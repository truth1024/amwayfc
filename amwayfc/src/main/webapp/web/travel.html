入境

必须持有效护照，并取得有效签证；
遵守海关的有关规定，特别是不得携带违禁品入关；
入境时自觉接受边检人员的例行检查。

时差

法国与中国的时差为6 小时。

法国退税的相关资讯

1. 购物者有资格享有退税购物
* 如果您拥有欧盟以外国家的永久居住权（您必须是一个非欧盟居民）。 * 如果您是15 岁以上。 2. 要在挂有-Tax Free Shopping、-Tax Refund或-Euro Free Tax等退税标识的商店里 购买的商品，才可以享受退税服务。
3. 最低购买金额：同一天在同一家商店内的购物金额达到或超过175.欧元。
4. 退税商品：
* 只有能够亲自携带（旅客个人行李）出境的商品，才可以申请退税； * 最多50 件相同的商品。 5. 不予退税的商品：烟草 / 药品 / 汽车及汽车零件 / 武器 / 文物 / 邮票

西班牙退税的相关资讯

1. 购物者有资格享有退税购物：
* 如果您拥有欧盟以外国家的永久居住权 （您必须是一个非欧盟居民）。 * 如果您是16 岁以上。 2. 最低购买金额：同一天在同一家商店内的购物金额达到或超过90.16 欧元。
3. 退税商品： * 只有能够亲自携带（旅客个人行李）出境的商品，才可以申请退税。 * 出口的商品必须是密封且未使用过的。

餐饮礼仪

1) 作为举世皆知的世界三大烹饪王国之一，法国人十分讲究饮食。在西餐之中，法国菜可以说是最讲究的。
2) 法国人爱吃面食，面包的种类很多；他们大都爱吃奶酪；在肉食方面，他们爱吃牛肉、猪肉、鸡肉、鱼子酱、鹅肝，不吃肥肉、宠物、肝脏之外的动物内脏、无鳞鱼和带刺骨的鱼。
3) 法国人特别善饮，他们几乎餐餐必，而且讲究在餐桌上要以不同品种的酒水搭配不同的菜肴；除酒水之外，法国人平时还爱喝生水和咖啡。
4) 法国人用餐时，两手允许放在餐桌上，但却不许将两肘支在桌子上，在放下刀叉时，他们习惯于将其一半放在碟子上，一半放在餐桌上。
